# 重学前端 - 汇总

## 📖 专题简介

系统性地重新学习前端核心知识，深入理解底层原理：

- **JavaScript 核心**: 作用域、闭包、原型链、this 指向等
- **异步编程**: 事件循环、Promise、async/await 原理
- **浏览器原理**: 渲染机制、重排重绘、性能优化
- **网络协议**: HTTP/HTTPS、TCP/IP、WebSocket
- **前端工程化**: 模块化、构建工具、性能优化
- **设计模式**: 常用设计模式在前端中的应用

## 📝 学习目标

- 夯实前端基础，理解而非记忆
- 掌握核心原理，提升问题解决能力
- 建立完整的前端知识体系
